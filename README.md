# bob-plugin-mimo-2-tts

这是一个 Bob 插件，使用小米 MiMo-2-TTS 模型将文本转换为语音。

## 前置条件

你需要 [API KEY](https://platform.xiaomimimo.com/#/console/api-keys) 才能使用这个插件。获取 API KEY 前需要完成以下步骤：

1. 注册并登录 [小米 MiMo 平台](https://platform.xiaomimimo.com/)
2. 完成实名认证
3. 充值至少 1 元 （目前使用是免费的）

## 使用方法

1. 下载插件，双击安装。
2. 打开 https://platform.xiaomimimo.com/#/console/api-keys 获取 API KEY。
3. 打开 Bob 的设置，找到 MiMo-2 TTS 服务，在 `API KEY` 中输入你的 API KEY。

## 支持语种

- 中文（简体/繁体）
- 英语
